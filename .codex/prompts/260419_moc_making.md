# 概要

# 機能一覧

## ギター登録機能

ギターの情報を登録する機能です。ギターの名前を登録することができます。

## 弦交換時期設定機能

弦交換の時期を設定する機能です。
ギターには6本の弦があり、それぞれの弦の交換時期を日付単位で設定することができます。

## 弦交換通知機能

弦交換の時期が近づいたときに通知する機能です。
ギターの各弦について、設定された交換時期が近づいたときに、ユーザに通知することができます。

## ユーザ設定機能

ユーザの設定を行う機能です。

- 弦交換の通知のタイミングを設定できます。例えば、交換時期の1週間前や3日前など、ユーザが希望するタイミングで通知を受け取ることができます。

# ユーザ操作フロー

1. ユーザはギター登録機能を使用して、ギターの名前を登録します。
2. ユーザは弦交換時期設定機能を使用して、各弦の交換時期を日付単位で設定します。
3. ユーザはユーザ設定機能を使用して、弦交換の通知のタイミングを設定します。
4. システムは弦交換の時期が近づいたときに、ユーザに通知します。

# 画面一覧

## 基本ルール

- SPA（Single Page Application）で構築されるため、画面遷移はなく、すべての機能が1つのページ内で完結します。
- 画面はシンプルで直感的なデザインを採用し、ユーザが簡単に操作できるようにします。
- ギター登録、弦交換時期設定、ユーザ設定の各機能は、ポップアップやモーダルウィンドウを使用して表示されます。
- 各ギターの情報はカード形式で表示され、ユーザが簡単に確認できるようにします。
